# BASB 核心概念速查

## 第二大脑定位与目标
- 第二大脑 = 个人化数字知识库，作用是“让信息为行动服务”，让灵感快速转成可执行成果。
- 关注三大价值：更快找到资料、更清楚优先级、更容易把知识转成成果。
- 《打造第二大脑实战手册》五大承诺（可用来衡量答复是否到位）：
  1. 停止浪费时间找资料——知道关键笔记在哪、几秒找到。
  2. 聚焦最重要的事情——生活兴趣与目标保持一致。
  3. 持续做出成果——有始有终、复用过往洞见加速进步。
  4. 创造力与生产力飙升——进入自建“创意游乐场”。
  5. 克服信息焦虑与 FOMO——相信自己拥有行动所需的全部材料。

## CODE 四步（抓取 Capture → 组织 Organize → 提炼 Distill → 表达 Express）
| 步骤 | 目的 | 常见卡点 | 提示问句 |
|------|------|----------|----------|
| 抓取 | 记录打动你的信息，优先“重磅”内容 | 试图抓取全部信息导致过载 | “这条信息何时能帮到我？若没有明确行动，不抓” |
| 组织 | 依据 PARA 按行动接近程度放置 | 沉迷按主题/来源分类 | “这条信息最先服务哪个项目/领域？” |
| 提炼 | 在内容上做荧光笔式处理，提炼最有力的 20% | 保留原文过多，复盘成本高 | “如果明天要引用，只需记住哪三句话？” |
| 表达 | 用第二大脑产出交付物（方案、输出） | 卡在“完美”或不知道下一步 | “最小可交付是什么？需要谁反馈？” |

## 常见 FAQ 模板
1. **为什么要区分抓取和组织？** 因为抓取发生在灵感出现的瞬间，组织需要更冷静的判断。先抓取可降低记录门槛，再用例行复盘/每周回顾来组织。
2. **如何缓解信息焦虑？** 回到五大承诺：当你知道资料可随时被找回，就能把注意力释放给当前项目。建议：限制抓取入口、每周清点 PARA、把无关内容移入存档。
3. **如何开始构建第二大脑？** 步骤：
   - 建立 PARA 四个顶层文件夹。
   - 选 1–2 个正在推进的项目，把相关内容集中进去。
   - 设定固定复盘时间（如周日 30 分钟）执行 CODE：检视新抓取→重排组织→提炼关键词→列出下一次表达机会。
4. **怎样判断一个领域？** 领域（Areas）是“持续责任”——没有明确完成点但需要维护标准，例如“职业发展”“健康”“家庭运营”。一旦该责任不再需要持续注意，就可以转存档。
